loadpath() click to toggle source

This is needed if a project has loadpaths other than the standard lib/. Note the routine is designed to handle YAML arrays or line by line list. 

# File lib/setup/project.rb, line 37
    def loadpath
      @loadpath ||= (
        if file = Dir.glob('{script/setup,meta,.meta}/loadpath').first
          raw = File.read(file).strip.chomp(']')
          raw.split(/[\n,]/).map do |e|
            e.strip.sub(/^[\[-]\s*/,'')
          end
        else
          nil
        end
      )
    end
